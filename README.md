# Zotonic Language Server

Zotonic support for [Language Server Protocol (LSP)](https://microsoft.github.io/language-server-protocol/).
No tests and no improvements were planned or made, only some ideas as code.

## Features

- Snippets
- Template search
- Go to template
- Docs integration
